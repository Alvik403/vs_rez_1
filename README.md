# Консолидация резервов

Сайт на FastAPI для загрузки Excel-файлов с листом `Вскрытие резервов` и сборки консолидированного файла.

## Запуск в Docker

```powershell
docker compose up --build
```

Откройте сайт: http://localhost:8000

## Как пользоваться

1. Выберите один или несколько файлов `.xlsx`/`.xlsm` с листом «Вскрытие резервов».
2. Нажмите `Собрать и скачать`.

## Итоговый файл

Скачивается **`.xlsx`** — новая книга с листами «Справка по резервам », «Справка», «СВОД общий».
